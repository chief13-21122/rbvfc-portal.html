What is LOSAP?

LOSAP (Length of Service Award Program) is a Maryland state benefit program that rewards active volunteer fire and EMS members for their service to the community.

Members who earn 50 or more points in a calendar year qualify for an annual benefit payment deposited into a tax-deferred retirement account.

How Points Are Earned
🚒 Emergency Response
1 pt each
🏛 Company Meeting
1 pt (max 10)
📋 Board Meeting
1 pt (max 10)
📚 Training / Drill
2 pts/hr (max 20)
🕐 Station Duty
3 pts/hr (max 25)
🏅 Elected Position
25 pts (max 25)

Questions? Contact your LOSAP Committee Chair or visit mfri.org for official program details.

CAD Gmail Sync Setup

5-step setup — no Google Cloud Console required.

Step 1 — Gmail Label & Filter
Create a Gmail label CAD-Dispatch and set up a filter to apply it to incoming CAD emails.
Step 2 — Google Sheet
Create a new Google Sheet. Copy the Sheet ID from the URL: docs.google.com/spreadsheets/d/SHEET_ID/edit
Step 3 — Apps Script
In the Sheet: Extensions → Apps Script. Paste the RBVFC-CAD-Sync.gs code, fill in your SHEET_ID, run once to grant permissions, then add a time-based trigger (every 30 minutes) for processDispatchEmails.
Step 4 — Deploy as Web App
Deploy → New Deployment → Web App. Execute as: Me. Who has access: Anyone. Copy the Web App URL.
Step 5 — Paste URL in Portal
Paste the Web App URL into Admin → Data Management → CAD Dispatch Sync and save.
